Lesson 2 - Creating structured documents

Front end web development is a very rewarding place to start programming because your results are visual.

Web pages are composed of three languages:

  • HTML - Content of the Page
  • CSS - Style or theme, also allows for a consistent look and feel
  • JS - Interactive code producing the functions of the page
  • HTML and CSS are both "languages"
  • HTML controls the "structure" of a web page
  • CSS controls the "style" of a page (how it looks).
  • When programmers talk about the "DOM" (Document Object Model) they are talking about the tree-like structure of a page

Steps to do when linking a stylesheet

  1. Add <!DOCTYPE html>
  2. Enclose all the rest of the code within <html> tags.
  3. Add a head enclosure to contain a link to the style sheet.
  4. Enclose the rest of the code within body tags.
Visual Basic will create this structure for you which is probably why we are using a simpler tool.
